## 🚀 Proyectos Destacados

### 🎟️ [TicketHub](https://github.com/dcuevasi/TicketHub)

Sistema **enterprise de tickets** full-stack.
Angular 20 + FastAPI con JWT, dashboard con **Chart.js**, búsqueda en tiempo real, dark mode y exportación CSV.
Arquitectura escalable con lazy loading y Material Design.

---

### 🌀 [Sistema-Sorteo](https://github.com/dcuevasi/Sistema-Sorteo)

Gestor de participantes y sorteos con **Django 5.2 + Vue 3 (Composition API)**.
Uso de **Celery/Redis** para tareas asíncronas, autenticación JWT y panel administrativo reactivo.
Diseñado con arquitectura **REST API escalable**.

---

### ✋ [Hand-Gesture](https://github.com/dcuevasi/Hand-Gesture)

Convierte tu cámara web en un controlador de mouse usando **MediaPipe + OpenCV**.
Control intuitivo y sin contacto mediante gestos de la mano.
Integración con **PyAutoGUI** y precisión optimizada (<100 ms).
